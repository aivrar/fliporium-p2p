Stats refresh when you reload the page. No JavaScript ping-back, no fingerprinting, no per-visit tracking. Live counts come from the signaling server, which only ever sees that a connection exists — never its contents.

What each number means

Downloads
How many times fliporium.exe has been fully downloaded from this site. Only complete transfers count — the crawlers and link-scanners that probe the file and disconnect early (a constant background hum on the public internet) are filtered out, as are obvious bot user-agents. So this number leans toward real installs rather than curiosity pings.
People connected now
How many devices are connected to the signaling server right this moment — i.e. people who currently have Fliporium open and joined to a room.
Rooms active now
How many separate rooms have at least one person in them right now. The signaling server counts rooms and connections to introduce peers; it never sees what's said inside them (that's peer-to-peer and encrypted).

What this page doesn't do

  • It does not log which IP visited which stat page.
  • It does not show per-device data of any kind.
  • It does not store cookies.
  • It does not load any third-party scripts.

See the privacy page for the full posture.
